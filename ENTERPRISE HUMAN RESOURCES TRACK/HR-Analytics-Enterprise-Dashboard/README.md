A comprehensive Microsoft Excel analytics dashboard that processes and visualizes enterprise HR data for 1,470 employees — delivering workforce metrics, demographic cohort analysis, attrition drilldowns, and interactive slicers for stakeholder reporting.

Project Overview
Domain	Human Resources Analytics / Excel BI
Tools Used	Microsoft Excel (Pivot Tables, Power Query, Charts, Slicers, VLOOKUP, Conditional Formatting)
Dataset	HR Data.xlsx — 1,470 employee records
Key Files	HR_Analytics_Dashboard.xlsx, HR_Analytics_Excel_Dashboard
Deliverable	Two polished interactive Excel HR dashboards

What I Did
•	Processed and structured the 1,470-employee dataset to isolate operational patterns across attrition demographics, department metrics, and job satisfaction tiers.: Workforce Metrics Auditing
•	Segmented employees into 8 distinct age groups — discovering a peak workforce concentration of 274 employees within the 32–35 age bracket.: Demographic Cohort Segmentation
•	Built distribution charts breaking down 237 total attrition departures — Bachelor's Degree holders (99) and R&D Department employees (133) represent the highest exit segments.: Attrition Drilldowns
•	Added Excel slicers enabling stakeholders to filter all dashboard visuals by Department, Gender, Job Role, and Education Level instantly.: Interactive Slicers
•	Used VLOOKUP, INDEX/MATCH, COUNTIFS, and nested IF formulas for dynamic metric calculations across pivot tables.: Advanced Excel Formulas
•	Applied color-coded conditional formatting to highlight attrition hotspots and satisfaction rating patterns across all employee segments.: Conditional Formatting

Key Insights
•	237 total attrition departures represent a 16.12% attrition rate — above the healthy industry benchmark of 10–12%.
•	R&D Department (133 departures) and employees with Bachelor's Degrees (99 departures) are the highest exit segments.
•	The 32–35 age bracket is the most concentrated employee group, making them a priority for engagement and retention.
•	Employees with low job satisfaction scores (1–2) are 3x more likely to leave than those with scores of 3–4.

Excel Features Used
•	Pivot Tables — for dynamic aggregation of workforce metrics by department, role, and age
•	Power Query — for data cleaning and transformation before dashboard build
•	Charts — bar charts, pie charts, line charts, and histograms for visual storytelling
•	Slicers — interactive filters for real-time dashboard updates
•	VLOOKUP / INDEX-MATCH — for cross-referencing employee attributes
•	Conditional Formatting — color-coded highlights for attrition risk and satisfaction levels
